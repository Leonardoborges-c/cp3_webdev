# CP3 WEBDEV

# Portfólio — Leonardo Borges da Costa

Este repositório tem meu portfólio pessoal de trabalho, aqui apresento linhas de códigos com rotas, lista de projetos com busca, apresentação pessoal e meu formulário de contato.
 O projeto foi feito como cp3 do curso de Web Dev.

## Funcionalidades usadas
- Menu de navegação com rotas.
- Página de apresentação pessoal.
- Lista de projetos com barra de busca, que filtra enquanto digita.
- Rotas dinâmicas para detalhes de cada projeto com: `/projetos/:id`.
- Formulário de contato que exibe os dados em um `alert` após enviar.
- Rodapé com ícones de redes sociais usando `react-icons`.
